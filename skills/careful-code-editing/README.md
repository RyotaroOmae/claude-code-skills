# careful-code-editing

AIコーディング時の典型的な失敗（早合点・過剰設計・範囲外の変更・検証不足）を
減らすための行動ルール集。

## 使い方

- **自動参照**: コードの実装・修正・リファクタリング・レビュー指摘への対応・テスト追加など、
  `description` に合致する「コードを書く/直す」作業のときに Claude Code が参照する
  （コードレビューそのものを行うスキルではない。レビュー実施は `/codex-review` 等を使う）。
- **明示呼び出し**: `/careful-code-editing` と入力すると、そのタスクの間ルールを適用する。
  対象ファイルや注意点を引数で渡してもよい（例: `/careful-code-editing src/foo.py の修正`）。

ルール本体は `SKILL.md` を参照。簡単な作業まで止めないための
「軽微な変更の例外」も SKILL.md 末尾に定義している。
